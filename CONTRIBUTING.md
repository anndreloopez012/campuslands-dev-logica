# Guia de contribucion

## Ramas

- No trabajes en `main`.
- No trabajes directo en `dev`.
- Crea ramas personales desde `dev`.

## Commits profesionales

Usa Conventional Commits:

- `feat(logica): resolver ejercicio 001`
- `fix(logica): corregir validacion de empate`
- `docs(entrega): explicar pruebas del ejercicio 010`
- `refactor(logica): separar calculo de salida`
- `test(logica): agregar casos borde`
- `chore(repo): actualizar estructura local`

## Entregas

Solo entrega dentro de:

```text
area/ejercicio-XXX/resoluciones/nombre-apellido/
```

No modifiques archivos base ni respuestas de otros estudiantes.
