# Instrucciones generales

Este repositorio esta disenado para entrenar pensamiento logico con ejercicios progresivos.

## Como estudiar

1. Lee el enunciado completo.
2. Escribe entradas, proceso y salida.
3. Resuelve un ejemplo a mano.
4. Programa o documenta la solucion.
5. Prueba con al menos dos casos.
6. Explica el resultado.

## Reglas de orden

- Trabaja desde una rama personal creada desde `dev`.
- Entrega solo dentro de `resoluciones/nombre-apellido/`.
- No cambies README base ni plantillas.
- No borres archivos de otros estudiantes.
- No abras PR hacia `main`.

## Recomendacion pedagogica

No busques resolver rapido. Busca resolver claro. Una solucion entendible vale mas que una solucion corta que nadie puede explicar.
