# Flujo de entrega

1. Haz fork del repositorio.
2. Clona tu fork.
3. Cambia a `dev`.
4. Crea una rama personal.
5. Trabaja solo dentro de tu carpeta en `resoluciones/nombre-apellido/`.
6. Haz commits claros.
7. Sube tu rama.
8. Abre PR hacia `dev`.

```bash
git switch dev
git pull origin dev
git switch -c alumno/nombre-apellido/ejercicio-001
git add .
git commit -m "feat(logica): resolver ejercicio 001"
git push origin alumno/nombre-apellido/ejercicio-001
```
